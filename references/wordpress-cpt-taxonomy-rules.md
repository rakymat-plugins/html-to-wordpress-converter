# WordPress CPT and Taxonomy Rules

Create a CPT only when the content needs one or more:

- repeated entries managed independently
- reuse across pages
- filtering/search
- archive pages
- single URLs
- admin workflow ownership

Do not create CPTs for small section-local cards, visual decorations, one-off logos, or simple icon lists. Use ACF repeaters or groups.

Taxonomies must support real grouping/filtering needs, not merely mirror section headings.
